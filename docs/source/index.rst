GBO 20m User Documentation Hub
==============================

The GBO 20m User Documentation Hub is a prototype documentation system under active development. It is meant to consolidate all user documentation to use GBO's 20m Telescope and process data obtained with the 20m Telescope.

-------------------

The hub follows the diataxis approach, where documentation is categorized with the user's need in mind. As a 20m user, you will be able to find Tutorials, How-to Guides, Explanations and Reference material that support your learning and the working aspects of your day-to-day tasks with either practical step-by-step instructions or knowledge-providing material.
